Saving as PDF
Open this file in a desktop browser → press P or click the PDF button → choose “Save as PDF” in the print dialog → paper size: 1280 × 720 (or A4 landscape) → background graphics: ON for color, OFF for a lighter print. Each slide will export as its own page.
