Appendix F — Audience View Reference

Use the toggle at the top of the deck. Same content surface, different disclosure.
Investor Preview (default)
  • Full strategic narrative
  • Product, market, business model and roadmap
  • Resources referred to by code only
  • No equity percentages, contracts, or partner names
This is the safe default for any external review.
Internal View
  • All of Investor Preview, plus
  • Strategic meaning behind each resource code
  • Internal-only operating notes
  • Still no real external partner names
For family, mentor, and risk-control review.
How to switch: Top of the deck → Investor Preview / Internal View toggle. Keyboard shortcut: I. The current view is reflected in real time across all slides.
