# RBW — Confidential Strategy Deck

**CONFIDENTIAL — DO NOT DISTRIBUTE.** 商业机密文件，未经书面许可不得复制、转发、上传或传播。

## 下载与使用

1. 点击 GitHub 仓库右上角 **Code → Download ZIP**。
2. 解压到桌面。
3. 双击 `RBW_Confidential_Strategy_Deck.html` 在浏览器中打开。
4. 保证 `slides.js`、`slides2.js` 与 HTML 文件在**同一文件夹**下。

## 文件清单

| 文件 | 说明 |
|---|---|
| `RBW_Confidential_Strategy_Deck.html` | 主文件。双击打开。含 slides 1–10 与全部 UI / CSS / JS。 |
| `slides.js` | Slides 11–25：GTM、Resources、Vibe Party、RBW Meta。 |
| `slides2.js` | Slides 26–43：Business Model、Roadmap、Metrics、Risk、Governance、Closing、Appendix A–G。 |

## 控制方式

| 操作 | 快捷键 |
|---|---|
| 下一页 | → / Space / PageDown / 点击幻灯右侧 |
| 上一页 | ← / PageUp / 点击幻灯左侧 |
| 首页 / 末页 | Home / End |
| 目录 | T 或顶部 Contents 按钮 |
| 关闭弹窗 | Esc |
| 切换 Internal / Investor View | I 或顶部切换按钮 |

| 移动端 | 左右滑动 |

## 导出 PDF

桌面浏览器打开 → 按 **P** 或点击右上 **PDF** 按钮 → 在系统打印对话框中选 **Save as PDF** → 纸张大小 1280×720 或 A4 横向 → 背景图形 ON → 保存。每张幻灯单独一页。

## 披露等级

- **Internal View** — 家族 / 商业师父 / 风控负责人可见完整战略逻辑，外部资源以代号展示。
- **Investor Preview （默认）** — 潜在投资人只看到战略、产品、市场、商业模型与路线图。
- **Post-NDA** — 签 NDA 后才披露具体资源、股权架构、财务细节。
