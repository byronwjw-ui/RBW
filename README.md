## 下载与使用 / Download & Use

1. 仓库右上角 **Code → Download ZIP**
2. 解压到桌面
3. 双击 `RBW_Confidential_Strategy_Deck.html` 在浏览器中打开
4. 确保 `slides.js`、`slides2.js`、`slides_zh.js`、`slides2_zh.js` 与 HTML 在**同一文件夹**

## PDF 导出 / Export as PDF

Desktop browser → press **P** or click **PDF** → choose "Save as PDF" → paper size **1280 × 720** or A4 landscape → background graphics **ON** → save. Each slide prints as its own page.

桌面浏览器 → 按 **P** 或点 **PDF** → 系统对话框选"另存为 PDF" → 纸张 **1280 × 720** 或 A4 横向 → 背景图形 **开** → 保存。每一页 slide 单独成页。
