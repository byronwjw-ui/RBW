## 三档披露 / Three Disclosure Tiers

切换后**整份 Deck 实时变化**——不只切语言，连内容显示都会变。

### Tier 1 · Investor Preview（默认）

- 给潜在投资人 / For potential investors
- 看到：战略、产品、市场、商业模型、路线图
- 看不到：资源含义、合作方上下文、股权细节

### Tier 2 · Internal

- 给创始团队 + 资深顾问 / For founders + senior advisors
- 看到：Investor 全部 + 每个 Resource 代号背后的战略含义 + 内部运营注释
- 看不到：尽调级细节、外部合作方实名

### Tier 3 · Post-NDA

- 给签 NDA 后的投资人 / For investors after NDA
- 看到：Internal 全部 + 资源推进阶段、股权结构概要、里程碑日历
- 看不到：仍受法律流程控制的事项；外部合作方真实名称

> 切换时**不刷新页面**。某些 slide（如 Resource Strategy、Company Structure）会动态露出额外的卡片块。

## PDF 导出 / Export as PDF

桌面浏览器打开 → 按 **P** 或点 **PDF** → 系统对话框选 "Save as PDF" → 纸张 **1280 × 720** 或 A4 横向 → 背景图形 **开** → 保存。每页 slide 单独成页。

> 提示：在导出前先把披露档调到你想发的版本（例如要发给投资人就保持 Investor Preview）。导出的 PDF 只包含当前档可见的内容。
